                                                                   Student Management System
       Project Overview
 The Student Management System is a MySQL-based database project designed to manage and organize student,  course, instructor, enrollment, angrade data efficiently.
 It includes multiple SQL features like Subqueries, Views, Triggers, Joins, and Stored Procedures to simulate real-world database operations in an educational environment.
 
                SQL Concepts Implemented
 Joins
Used to combine data from multiple tables such as showing students with their courses and grades.

Subquery
Used to find students not enrolled in any course.

View
Created for simplified reporting combining student, course, and grade data.

Trigger
Automatically inserts a default grade record when a student is enrolled

Stored Procedure
Used to enroll a student safely and prevent duplicate entries.
